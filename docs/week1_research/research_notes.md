# Week 1 竞品视觉调研笔记

**日期**: 2026-01-18
**记录人**: 冯小婷

## 1. 调研目标
分析主流节点编排工具的交互逻辑与视觉风格，为 DoraMate 确立设计基调。

## 2. 竞品分析

### A. Node-RED (工业物联网流)
- **视觉风格**: 纯功能导向，配色较为杂乱（高饱和度色块），缺乏美感。
- **交互痛点**: 
    - 节点拥挤时，连线交叉严重，难以辨识数据流向。
    - 侧边栏图标陈旧，缺乏现代化的 UI 规范。
- **结论**: DoraMate 需避免这种“工程师审美”，我们要更轻量。

### B. Unreal Engine Blueprints (虚幻引擎蓝图)
- **视觉风格**: 深色模式 (Dark Mode)，线条平滑流畅，节点带有明确的输入/输出端口颜色区分。
- **优点**: 
    - "无限画布"的拖拽体验极佳。
    - 连线在选中时有高亮动画（流光效果），非常符合“具身智能”的实时感。
- **结论**: **重点参考对象**。我们将借鉴其“节点端口分类”和“动态连线”的设计。

## 3. DoraMate 设计方向定调
基于以上调研，确立 DoraMate V1.0 的设计关键词：
1.  **极简 (Minimalism)**: 去除多余装饰，让画布成为绝对主角。
2.  **科技感 (Tech-savvy)**: 默认采用深色模式，搭配高对比度的强调色（如荧光绿/蓝）。
3.  **可视化 (Visual feedback)**: 每一个数据流的传输，都应在界面上给予用户反馈。
