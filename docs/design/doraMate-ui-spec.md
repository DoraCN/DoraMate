# DoraMate Web UI 设计规格

## 1. 整体布局

页面分为四个区域：

1. 顶部工具栏 Toolbar（高度 56px）
2. 左侧节点面板 NodePanel（宽度 260px）
3. 中间画布区域 Canvas（自适应填充）
4. 右侧属性面板 PropertyPanel（宽度 300px）

整体背景色：#F3F4F6
主要卡片背景：#FFFFFF

布局方式：左右三栏 + 顶部固定工具栏
## 2. 工具栏 Toolbar

高度：56px  
背景色：#1F2937  
文字颜色：#FFFFFF  

包含按钮：
- 打开
- 保存
- 导入 YAML
- 导出 YAML
- 验证
- 本地运行（主按钮）

按钮样式：
- 普通状态：透明背景，白色文字
- 悬停状态：背景 #374151
- 主按钮（本地运行）：
  - 背景 #22C55E
  - 悬停背景 #16A34A
  - 圆角 6px
## 3. 节点面板 NodePanel

背景色：#F9FAFB  
节点卡片样式：
- 尺寸：220×56px
- 圆角：12px
- 背景：#FFFFFF
- 阴影：0 2px 8px rgba(0,0,0,0.1)

节点类型颜色标识（左侧色条）：
- 输入节点：#3B82F6（蓝）
- 处理节点：#22C55E（绿）
- 输出节点：#A855F7（紫）
- 错误状态：#EF4444（红）

交互：
- 悬停：整体背景变为对应颜色的浅色高亮
- 可拖拽到画布
## 4. 画布中的节点 Node

尺寸：180×110px  
圆角：16px  
背景：#FFFFFF  
阴影：0 4px 16px rgba(0,0,0,0.15)

顶部标题栏按类型着色（蓝/绿/紫）

选中状态：
- 外框描边 2px #2563EB

端口：
- 左侧为输入端口，右侧为输出端口
- 圆形，直径 10px，颜色 #9CA3AF
## 5. 连线样式 Edge

默认：
- 颜色：#6B7280
- 宽度：2px
- 贝塞尔曲线

合法连接预览：
- 颜色：#3B82F6
- 虚线

错误连接：
- 颜色：#EF4444

运行中动画：
- 颜色：#22C55E
- 连线上有流动的小圆点动画
## 6. 属性面板 PropertyPanel

宽度：300px  
背景：#FFFFFF  
标题：节点属性

表单项间距：12px

高级选项默认折叠：
- 点击“高级设置”后展开
## 7. 即时反馈与动效

- 按钮点击：背景颜色加深
- 节点拖拽中：透明度 0.7
- 非法连接时：连线立即变红
- 本地运行后：连线出现绿色流动动画
- 所有状态切换使用 0.2s 过渡动画
