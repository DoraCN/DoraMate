## 🔄 12.5 数据转换器

### YAML ↔ 可视化图 转换

**文件**: `doramate-frontend/src/utils/converter.rs`

```rust
use crate::types::{Dataflow, DoraDataflow, Node, Connection, DoraNode, DoraMateMeta, LayoutInfo};
use std::collections::HashMap;

/// 可视化图 → DORA YAML
impl From<&Dataflow> for DoraDataflow {
    fn from(dataflow: &Dataflow) -> Self {
        let mut layout = HashMap::new();
        let mut nodes = Vec::new();

        // 转换节点
        for node in &dataflow.nodes {
            // 保存布局信息
            layout.insert(
                node.id.clone(),
                LayoutInfo {
                    x: node.x,
                    y: node.y,
                    label: Some(node.label.clone()),
                },
            );

            // 推断 DORA path 和 build
            let (path, build) = infer_node_path_and_build(&node.node_type);

            // 构建 DORA 节点
            let mut dora_node = DoraNode {
                id: node.id.clone(),
                path: Some(path),
                build,
                inputs: None,
                outputs: node.outputs.clone(),
                env: node.env.clone(),
                operators: None,
            };

            // 处理输入连接
            let mut inputs = HashMap::new();
            for conn in &dataflow.connections {
                if conn.to == node.id {
                    let from_port = conn.from_port.as_deref().unwrap_or("out");
                    let to_port = conn.to_port.as_deref().unwrap_or("in");
                    inputs.insert(to_port.to_string(), format!("{}:{}", conn.from, from_port));
                }
            }

            if !inputs.is_empty() {
                dora_node.inputs = Some(inputs);
            }

            nodes.push(dora_node);
        }

        DoraDataflow {
            __doramate__: Some(DoraMateMeta {
                layout: Some(layout),
                name: None,
                description: None,
                tags: None,
            }),
            nodes,
        }
    }
}

/// DORA YAML → 可视化图
impl From<&DoraDataflow> for Dataflow {
    fn from(dora_dataflow: &DoraDataflow) -> Self {
        let layout = dora_dataflow
            .__doramate__
            .as_ref()
            .and_then(|meta| meta.layout.clone());

        let mut nodes = Vec::new();
        let mut connections = Vec::new();

        // 转换节点
        for dora_node in &dora_dataflow.nodes {
            let layout_info = layout.as_ref()
                .and_then(|l| l.get(&dora_node.id));

            let (node_type, label) = if let Some(info) = layout_info {
                let node_type = infer_node_type_from_path(
                    dora_node.path.as_deref().unwrap_or("")
                );
                (node_type, info.label.clone().unwrap_or_else(|| dora_node.id.clone()))
            } else {
                let node_type = infer_node_type_from_path(
                    dora_node.path.as_deref().unwrap_or("")
                );
                (node_type, dora_node.id.clone())
            };

            let node = Node {
                id: dora_node.id.clone(),
                x: layout_info.map(|l| l.x).unwrap_or(0.0),
                y: layout_info.map(|l| l.y).unwrap_or(0.0),
                label,
                node_type,
                path: dora_node.path.clone(),
                env: dora_node.env.clone(),
                config: None,
                outputs: dora_node.outputs.clone(),
                inputs: dora_node.inputs.as_ref()
                    .map(|m| m.keys().cloned().collect()),
                scale: None,
            };

            nodes.push(node);

            // 提取连接关系
            if let Some(inputs) = &dora_node.inputs {
                for (to_port, source) in inputs {
                    if let Some((from, from_port)) = parse_source(source) {
                        connections.push(Connection {
                            from,
                            to: dora_node.id.clone(),
                            from_port: if from_port == "out" { None } else { Some(from_port.to_string()) },
                            to_port: if to_port == "in" { None } else { Some(to_port.clone()) },
                        });
                    }
                }
            }
        }

        Dataflow { nodes, connections }
    }
}
```

```rust
/// 验证数据流
pub fn validate_dataflow(dataflow: &Dataflow) -> Vec<ValidationError> {
    let mut errors = Vec::new();

    // 检查节点 ID 唯一性
    let mut ids = std::collections::HashSet::new();
    for node in &dataflow.nodes {
        if !ids.insert(&node.id) {
            errors.push(ValidationError {
                code: "DUPLICATE_NODE_ID".to_string(),
                message: format!("重复的节点 ID: {}", node.id),
                severity: ErrorSeverity::Error,
            });
        }
    }

    // 检查连接的有效性
    for conn in &dataflow.connections {
        let from_exists = dataflow.nodes.iter().any(|n| n.id == conn.from);
        let to_exists = dataflow.nodes.iter().any(|n| n.id == conn.to);

        if !from_exists {
            errors.push(ValidationError {
                code: "SOURCE_NODE_NOT_FOUND".to_string(),
                message: format!("源节点不存在: {}", conn.from),
                severity: ErrorSeverity::Error,
            });
        }

        if !to_exists {
            errors.push(ValidationError {
                code: "TARGET_NODE_NOT_FOUND".to_string(),
                message: format!("目标节点不存在: {}", conn.to),
                severity: ErrorSeverity::Error,
            });
        }
    }

    // 检查循环依赖
    if has_cycles(dataflow) {
        errors.push(ValidationError {
            code: "CYCLIC_DEPENDENCY".to_string(),
            message: "数据流存在循环依赖".to_string(),
            severity: ErrorSeverity::Error,
        });
    }

    errors
}

/// 验证错误
#[derive(Clone, Debug)]
pub struct ValidationError {
    pub code: String,
    pub message: String,
    pub severity: ErrorSeverity,
}

/// 错误严重程度
#[derive(Clone, Debug, PartialEq, Eq)]
pub enum ErrorSeverity {
    Warning,
    Error,
}

/// 检查是否有循环依赖
fn has_cycles(dataflow: &Dataflow) -> bool {
    // 使用 DFS 检测环
    use std::collections::{HashSet, VecDeque};

    let mut graph = std::collections::HashMap::new();
    for node in &dataflow.nodes {
        graph.entry(&node.id).or_insert_with(Vec::new);
    }

    for conn in &dataflow.connections {
        graph.entry(&conn.from)
            .or_insert_with(Vec::new)
            .push(&conn.to);
    }

    let mut visited = HashSet::new();
    let mut rec_stack = HashSet::new();

    for node_id in graph.keys() {
        if dfs_check_cycle(node_id, &graph, &mut visited, &mut rec_stack) {
            return true;
        }
    }

    false
}

fn dfs_check_cycle(
    node_id: &str,
    graph: &std::collections::HashMap<&String, Vec<&String>>,
    visited: &mut HashSet<&str>,
    rec_stack: &mut HashSet<&str>,
) -> bool {
    visited.insert(node_id);
    rec_stack.insert(node_id);

    if let Some(neighbors) = graph.get(&node_id.to_string()) {
        for neighbor in neighbors {
            if !visited.contains(neighbor.as_str()) {
                if dfs_check_cycle(neighbor, graph, visited, rec_stack) {
                    return true;
                }
            } else if rec_stack.contains(neighbor.as_str()) {
                return true;
            }
        }
    }

    rec_stack.remove(node_id);
    false
}
```

```rust
    fn validate(&self) -> Result<(), validator::ValidationErrors> {

        if let Err(e) = serde_yaml::from_str::<serde_yaml::Value>(&self.dataflow_yaml) {

                "dataflow_yaml",
                ValidatorError::new(format!("YAML 格式错误: {}", e)),
            );
        }

        if errors.is_empty() {
            Ok(())
        } else {
            Err(errors)
        }
    }
}
```

```rust
/// 文件系统管理器
pub struct FileSystemManager {
    base_dir: std::path::PathBuf,
}

impl FileSystemManager {
    /// 打开数据流文件
    pub async fn open_dataflow(&self, path: &std::path::Path) -> Result<Dataflow, IoError> {
        let content = tokio::fs::read_to_string(path).await?;
        let dora_dataflow: DoraDataflow = serde_yaml::from_str(&content)?;
        let dataflow = Dataflow::from(&dora_dataflow);
        Ok(dataflow)
    }

    /// 保存数据流文件
    pub async fn save_dataflow(
        &self,
        path: &std::path::Path,
        dataflow: &Dataflow
    ) -> Result<(), IoError> {
        // 自动备份
        if self.config.auto_backup {
            self.create_backup(path).await?;
        }

        // 转换为 DORA 格式
        let dora_dataflow = DoraDataflow::from(dataflow);

        // 序列化为 YAML
        let yaml = serde_yaml::to_string(&dora_dataflow)?;

        // 写入文件
        tokio::fs::write(path, yaml).await?;

        // 更新最近文件
        self.add_to_recent(path).await?;

        Ok(())
    }

    /// 添加到最近文件列表
    async fn add_to_recent(&self, path: &std::path::Path) -> Result<(), IoError> {
        let recent_path = self.base_dir.join("recent.json");

        // 读取现有列表
        let mut recent_files = if recent_path.exists() {
            let content = tokio::fs::read_to_string(&recent_path).await?;
            serde_json::from_str::<RecentFiles>(&content)?
        } else {
            RecentFiles { recent: Vec::new() }
        };

        // 添加新条目
        let entry = RecentFileEntry {
            path: path.to_string_lossy().to_string(),
            opened_at: chrono::Utc::now().to_rfc3339(),
        };

        recent_files.recent.insert(0, entry);

        // 限制列表长度
        recent_files.recent.truncate(20);

        // 保存
        let content = serde_json::to_string_pretty(&recent_files)?;
        tokio::fs::write(&recent_path, content).await?;

        Ok(())
    }
}
```

## 📊 12.8 数据模型对比

### 与 Blazor/C# 版本对比

| 数据结构 | Rust 版本 | Blazor 版本 | 优势 |
|---------|----------|------------|------|
| **节点** | `Node` (Struct) | `GraphNode` (Class) | Rust: 零成本抽象、栈分配 |
| **连接** | `Connection` (Struct) | `Connection` (Class) | Rust: 不可变、线程安全 |
| **状态** | `NodeState` (Enum) | `NodeStatus` (Enum) | Rust: 模式匹配、编译时检查 |
| **元数据** | `Option<T>` | nullable | Rust: 编译时空值检查 |
| **序列化** | `serde` | `System.Text.Json` | Rust: 零拷贝、类型安全 |
| **验证** | `validator` + 手动 | `DataAnnotations` | Rust: 编译时验证、零开销 |

### 纯文件系统 vs 数据库对比 ⭐

| 维度 | 纯文件系统 (Rust 版) | 数据库 (Blazor 版) | 优势 |
|-----|---------------------|-------------------|------|
| **配置复杂度** | ⭐⭐⭐⭐⭐ (零配置) | ⭐⭐ (需要 PostgreSQL) | **文件系统** |
| **依赖** | ⭐⭐⭐⭐⭐ (仅 serde_yaml) | ⭐⭐ (SeaORM + 驱动) | **文件系统** |
| **备份** | ⭐⭐⭐⭐⭐ (复制文件夹) | ⭐⭐⭐ (需要工具) | **文件系统** |
| **可读性** | ⭐⭐⭐⭐⭐ (YAML 人性化) | ⭐⭐ (需要查询) | **文件系统** |
| **版本控制** | ⭐⭐⭐⭐⭐ (Git 友好) | ⭐⭐⭐ (需要迁移) | **文件系统** |
| **查询性能** | ⭐⭐⭐ (线性扫描) | ⭐⭐⭐⭐⭐ (索引) | **数据库** |
| **并发安全** | ⭐⭐⭐⭐ (文件锁) | ⭐⭐⭐⭐⭐ (事务) | **数据库** |
| **扩展性** | ⭐⭐⭐ (单机) | ⭐⭐⭐⭐⭐ (分布式) | **数据库** |

**为什么选择纯文件系统?** ⭐:
- ✅ **零配置**: 用户无需安装、配置数据库
- ✅ **简化开发**: 专注核心功能,加速 MVP
- ✅ **可读性**: YAML 文件可直接阅读和编辑
- ✅ **版本控制**: Git 友好,易于协作
- ✅ **符合 DORA 哲学**: YAML 即配置
- ✅ **降低学习曲线**: 文件系统比数据库更直观

## 🎯 12.9 最佳实践

### 数据结构设计原则

1. **优先使用 Struct 而不是 Enum**
   - Struct 提供更灵活的扩展性
   - Enum 用于固定的状态类型

2. **使用 Option<T> 表示可选字段**
   - 避免 null 值
   - 编译时保证类型安全

3. **使用 HashMap 存储动态配置**
   - `HashMap<String, String>` 用于环境变量
   - `HashMap<String, serde_yaml::Value>` 用于动态配置

4. **添加 `#[serde(skip_serializing_if = "Option::is_none")]`**
   - 避免 YAML 中出现大量 null 值
   - 保持输出简洁

5. **使用 `#[serde(rename = "type")]` 处理关键字**
   - 避免与 Rust 关键字冲突
   - 保持 YAML 格式标准

### 数据转换最佳实践

1. **实现 `From<T>` Trait**
   - 提供零成本转换
   - 类型安全保证

2. **使用 `Option` 处理可选字段**
   - 避免 panic
   - 优雅降级

3. **提供合理的默认值**
   - 使用 `infer_*` 函数推断
   - 保持向后兼容

4. **验证数据完整性**
   - 检查循环依赖
   - 验证连接有效性
